# RNOR Calculator - Web Deployment Package

This folder contains everything you need to deploy the RNOR Calculator to GitHub Pages, Netlify, or any web hosting service.

## 📁 What's Included

- `index.html` — Main calculator page
- `rnor-logic.jsx` — Calculation engine
- `rnor-components.jsx` — UI components
- `assets/` — Turtle Finance logos

## 🚀 Quick Deploy

### GitHub Pages:
```bash
cd deployment
git init
git add .
git commit -m "Deploy RNOR Calculator"
git remote add origin https://github.com/YOUR-USERNAME/rnor-calculator.git
git push -u origin main
```
Then enable Pages in repo settings.

### Netlify:
1. Drag the `deployment` folder to netlify.com/drop
2. Done! Get instant live URL.
